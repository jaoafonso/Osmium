O Que é o Osmium?

O Osmium é uma rede social desenvolvida para gamers que possuem dificuldade em encontrar um parceiro ideal para jogos. Esse projeto foi desenvolvido para ser entregue como TCC no Curso Técnico de Desenvolvimento de Sistemas do SENAI mas continua sendo aprimorado até hoje.
